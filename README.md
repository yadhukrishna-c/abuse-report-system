# AI-Enabled Abuse Reporting & Counselling Support System

A secure platform for anonymous abuse reporting with AI-based support.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MySQL
- AI: Sentiment Analysis

## Features
- Anonymous abuse reporting
- AI-based sentiment analysis
- Real-time counselling chat
- SOS alert system
- Role-based access control

## Installation
### Backend
cd "abuse backend/abuse backend/express"
npm install
npm start

### Frontend
cd abuse
npm install
npm start
